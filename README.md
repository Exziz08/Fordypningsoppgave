# Fordypningsoppgave | PostgreSQL |
Om oppgaven

I denne fordypningsoppgaven har jeg jobbet med å lage en enkel webapplikasjon med postgreSQL. Målet mitt var å forstå hvordan denne typen database fungerer sammen med backend og frontend i et system. Jeg valgte å bruke PostgreSQL fordi det er en mye brukt med databaser , og koblet den til en backend laget i Node.js.

 # Hva jeg har lært

Gjennom arbeidet med prosjektet har jeg lært hvordan man oppretter en database og lager tabeller ved hjelp av SQL. Jeg har også lært hvordan man legger inn data og henter det ut igjen. Det viktigste jeg har lært er hvordan backend kan hente data fra databasen og sende det videre til frontend.

Jeg har fått en bedre forståelse av hvordan hele systemet henger sammen, der frontend sender forespørsler til backend, og backend kommuniserer med databasen før den sender et svar tilbake

# Utfordninger jeg møtte på veien

Jeg møtte flere problemer underveis. I starten fikk jeg ikke tilgang til PostgreSQL i terminalen fordi psql ikke fungerte, så jeg måtte bruke pgAdmin i stedet. Jeg hadde også problemer med å koble backend til databasen, spesielt med passord og tilkobling.
